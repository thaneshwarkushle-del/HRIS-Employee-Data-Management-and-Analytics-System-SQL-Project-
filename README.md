# HRIS-Employee-Data-Management-and-Analytics-System-SQL-Project-
🔹 What it is

A Human Resource Information System (HRIS) built with SQL to manage employee data and generate HR insights.

🔹 What it does

Stores employee details (name, role, department, salary, attendance).

Runs queries to analyze workforce trends.

Helps HR teams make data-driven decisions.

🔹 Tech Used

MySQL for database

SQL queries for analysis

🔹 Key Features

Clean and organized employee records

Department-wise headcount and salary analysis

Attrition and retention insights

Diversity and workforce metrics

🔹 Why it matters

This project shows how SQL can solve real HR problems by turning raw employee data into clear insights for management.

🔹 Learning

Through this project, I learned:

How to design normalized databases for HR systems.

Writing complex SQL queries for analytics.

Applying business storytelling with query results.

Building recruiter-facing projects that demonstrate practical problem-solving.
